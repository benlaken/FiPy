class fispy.fispy.Asset(**kwargs)[source]¶

Bases: object

Asset items are essentially dictionaries, and should be of a kind = ‘real_estate’, ‘stocks’, ‘job’, or ‘cash’. There can be infite assets in a Portfolio, but there should be only one cash Asset, which should specify a maximum amount of cash to hold.

monthly_repay()[source]¶

Examines a flag (asset.pay_debt_asap) from asset objects with debt. If an asset has the flag set to True, it will take as much as possible from the montly avaiable money (after expenses account for), and spend it on minimizing the debt. The minimum repayments should be covered for all assets before additional payments are considered.
